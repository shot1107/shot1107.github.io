ホーム

概要

Binary Indexed Tree, Fenwick(フェニック) Tree とも。
先頭からの累積和と１点加算のクエリを処理するデータ構造。さらにその情報から区間和も求めることができる。(sum[l, r] = sum[0, r]-sum[0, l-1])
アーベル群をのせることができる。ただし、先頭からの累積和だけなら可換モノイド(結合律、単位元の存在、交換律)で十分。
Segment Tree の下位互換のように思えるが、実装が軽い、定数倍速い等の利点もある。
